# <配置>Hy的安装与使用
## 安装Hy
```
$ pip install git+https://github.com/hylang/hy.git
```
## 使用
文件中添加

```
#! /usr/bin/env hy
```

然后执行

```
chmod +x Name.hy
./Name.hy
```
